mode - note

mode note is a visualiser for musical notes and scales for fretboards of extended range and standard guitars and basses.

features:
choose an instrument and the number of strings,
change the tuning of the full fretboard or each individual string independently,
view musical note names or interval positions,
flip between vertical and horizontal view,
toggle sharps or flats,
choose from a variety of musical scales or modes with any tonic value,

metronome!
change the bpm from 40 - 400,
choose from a list of different time signatures,
choose between 2 sounds (for now),
toggle on/off sound and light flash.
